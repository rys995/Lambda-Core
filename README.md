# Lambda Core

*A fan-made, non-commercial crossover project built around environmental storytelling, atmospheric pacing, and experimental narrative structure.*

Lambda Core is a fan-made **Touhou Project × Half-Life** crossover archive built around a simple but dangerous narrative question:

_**“What if the bad ending of Half-Life was not the end of Gordon Freeman… but instead, G-Man discarded him into Gensokyo?”**_

Rather than focusing on pure power scaling or "who is stronger", Lambda Core mainly revolves around the catalytic effect of Freeman himself — especially how his existence, survival instinct, fragmented experiences, and messianic reputation slowly affect the emotional and social ecosystem of Gensokyo itself.

This repository contains the current raw script archive (~266k words and still growing), written primarily as a narrative and gameplay-oriented foundation before gradually evolving into an actual playable project.

---

# Current Development Status

Lambda Core is currently under active development as a **Source Engine** fan project using **Mapbase (Source SDK 2013)** as its technical foundation.

The current development focus is building a playable prototype of the opening chapter, **Unknown Guest**, emphasizing:

- environmental storytelling,
- atmospheric exploration,
- scripted sequences,
- immersive pacing,
- and character-driven narrative progression.

The playable adaptation is being developed gradually alongside the existing raw scripts rather than replacing them.

---

# Notes

## 1. This crossover is **NOT** focused on power scaling.

The primary focus is atmosphere, character interaction, factional tension, psychological implications, and how Freeman's presence gradually changes the worldview of those around him.

---

## 2. The Source Engine adaptation will **NOT** be identical to the raw scripts.

Although the raw scripts serve as the project's primary narrative foundation, the playable version will naturally differ.

Gameplay pacing, environmental storytelling, player immersion, exploration flow, technical limitations, and chapter structure require many scenes, conversations, and narrative beats to be condensed, expanded, merged, or restructured into a format that works as an interactive experience.

The raw scripts should therefore be viewed as:

> *"The thematic and narrative foundation of Lambda Core rather than a direct 1:1 game storyboard."*

Both the scripts and the playable adaptation are expected to continue evolving throughout development.

---

## 3. Contributions are welcome.

If you are interested in contributing, feel free to use GitHub's contribution features.

All major additions, revisions, or creative proposals will still be reviewed carefully to maintain consistency regarding:

- Touhou characterization,
- Half-Life lore,
- atmosphere,
- thematic direction,
- continuity,
- and the overall narrative intent of Lambda Core.

---

## 4. Please document revisions clearly.

If you contribute changes to the scripts or project materials, please clearly mention:

- what was changed,
- which chapter or section was affected,
- and the purpose behind the revision.

Revision notes may be submitted through:

- Pull Requests,
- GitHub Issues,
- changelog files,
- markdown documentation,
- or dated revision logs explaining the applied changes and their reasoning.

This helps preserve long-term maintainability, interpretation clarity, and narrative consistency as the project continues to grow.

---

Some parts of Lambda Core intentionally rely on implication, fragmented perspectives, environmental storytelling, and emotional atmosphere instead of direct exposition.

If certain scenes, concepts, timelines, or narrative decisions feel confusing, feel free to ask.

Discussion, interpretation, and constructive feedback are always welcome.

---

Lambda Core originally started as a small passion project created during spare time.

Over time, it gradually evolved into a much larger long-term creative project than originally intended, eventually expanding beyond raw narrative writing into an actively developed Source Engine experience.

Thank you for taking the time to read, interpret, and explore this strange little archive.
